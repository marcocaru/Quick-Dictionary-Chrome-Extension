# Quick Dictionary

A Chrome extension that transforms how you read on the web. Highlight any word 
on any page to instantly surface its definition, part of speech, phonetic 
pronunciation, and audio playback — all in a polished Material You dark panel.

## Features

- Highlight any word to look it up instantly
- Click any word inside a definition to look that up too
- Breadcrumb trail lets you navigate your full lookup history
- Audio pronunciation with animated playback button
- Skeleton loading state and smooth enter/exit animations
- Smart positioning — popup always stays on screen
- Press Escape or click away to dismiss

## Stack

- Vanilla JS content script (no dependencies)
- Free Dictionary API (no key required)
- Material You / Google M3 dark color system
- Google Sans typography

## Installation

1. Clone or download this repo
2. Go to `chrome://extensions` and enable Developer Mode
3. Click **Load unpacked** and select the project folder
